# 010 Follower Merge

Status: future  
Priority: later  
Milestone: M2

## Outcome

Admin can compare a source list with the target francophone pack and import missing follows safely.

## User Stories

- `US-MERGE-01` Compare source and target packs
- `US-MERGE-02` Select members to import

## Acceptance Criteria

- Source and target can be loaded and compared.
- Importable, already-present, and target-only members are clearly separated.
- Import flow is non-destructive by default.
- Credits to Calle and Following.space are visible.
